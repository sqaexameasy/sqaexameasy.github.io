Introduction

In the following slides I will help you learn about the national 5 Computing Science course and how to pass the exam. The format will be the question, then variables if there is then -> and the answer, example 8 + 9 -> 17

Converting Numbers into 8-bit Binary: 1 mark

So far in all three exams there has been a question about converting a number into 8 bit binary. The numbers used have been 47, 164 and 227, with the answers are 47 -> 00101111, 164 -> 10100100 and 227 -> 11100011. If you got this wrong, don't fret! I found this instructables article to be very useful for learning it.

Hyperlinks: 1 or 2 marks

In 2/3 of the exams there has been a question about hyperlinks, A hyperlink provides navigation through a website or allows you to move to another resource (page). The difference between a internal and external hyperlink is that an internal hyperlink points to another page in the same website and an external hyperlink links to a different hyperlink.
